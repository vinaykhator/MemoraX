# Media Delivery Reference

## Video & Audio Assets (Enterprise Acquisition Package)

This repository does **not** contain any of the production video or audio assets used
within the MemoraX brand system.  
All media files are delivered exclusively as part of the  
**MemoraX Enterprise Acquisition Package (Verified Transfer – 2025).**

---

## 1. Purpose of This Document

This document explains how the MemoraX media suite is distributed and how it
relates to the contents of this repository.

All production-ready video and audio materials are included in the private
**Enterprise Acquisition Package**, not in the source-code repository.

These assets are intentionally excluded due to:

- file size and storage constraints  
- acquisition confidentiality  
- professional media delivery standards  
- separation of source code from production media  

---

## 2. Media Suite Contents

The media component of the  
**MemoraX Enterprise Acquisition Package (Verified Transfer – 2025)** includes:

- ProRes 422 HQ master video  
- H.264 20 Mbps master export

- WAV audio master  
- media-specific handover documentation  
- MD5 checksum integrity validation

## 3. Access & Distribution

All media assets are delivered directly to qualified parties via:

- secure ZIP distribution  
- encrypted transfer link  
- direct acquisition workflow  

They are not stored or mirrored in this repository.
