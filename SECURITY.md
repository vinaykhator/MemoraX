# Security Policy

## Supported Versions

This repository is provided exclusively for the purpose of reviewing,
evaluating, or negotiating a potential acquisition of the MemoraX brand
and its associated assets.

It is **not** an open-source project and does **not** accept:

- external code contributions  
- community patches  
- third-party security fixes  
- public issue reports  

All content is proprietary and controlled by the owner.

---

## Reporting a Vulnerability

If you believe you have identified a security-related concern involving:

- the MemoraX website  
- brand documentation  
- included assets  
- verification files  
- or any other material contained within this repository  

please contact:

**Email:** hello@memorax.ai  
**Website:** https://memorax.ai  

Security-related reports will be reviewed privately and only in the context
of acquisition evaluation.  
No public disclosures are permitted.
