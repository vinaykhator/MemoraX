# Support Policy

## How to Get Support

This repository is part of the MemoraX enterprise acquisition package.
Because it is not an open-source project, traditional community support
channels do not apply.

For all inquiries related to:

- acquisition  
- technical questions  
- brand assets  
- verification files  
- documentation  
- transfer process  

please contact:

**Email:** hello@memorax.ai  
**Website:** https://memorax.ai  

Support is provided exclusively for potential buyers or evaluators of the
MemoraX brand and identity system.

---

## Scope of Support

Support covers questions regarding:

- the contents of this repository  
- included brand assets  
- deployment structure  
- metadata and verification  
- brand documentation  

Support **does not** include custom development, implementation,
modification, or integrations unless explicitly negotiated as part of an
acquisition agreement.
